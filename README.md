# 税务记账助手

## 功能简介

本项目目前是一套面向个体工商户和个人独资企业的本地记账与税额计算工具，并提供企业信息查询辅助。当前版本**不包含**向税务局自动提交申报的能力。

### 主要功能

- **经营主体管理**：支持添加、编辑、删除经营主体
- **记账管理**：按日期记录收入与支出，并自动生成季度汇总
- **快速收入录入**：可快速录入季度收入和费用，数据优先读取记账汇总
- **税额自动计算**：
  - 增值税（小规模纳税人 / 一般纳税人）
  - 个人所得税（经营所得）：五级超额累进税率
  - 附加税（城建税 + 教育费附加 + 地方教育附加）
- **企业信息查询辅助**：通过浏览器查询并辅助提取企业基础信息

### 支持税种

| 税种 | 申报周期 | 说明 |
|------|----------|------|
| 增值税 | 月/季 | 小规模纳税人季度30万免税 |
| 个人所得税（经营所得） | 季/年 | 季度预缴，年度汇算清缴 |
| 附加税 | 随增值税 | 城建税7% + 教育费附加3% + 地方教育附加2% |


## Web 版本部署

除了桌面版本，本项目还提供 Web 版本，可以部署到服务器上通过浏览器访问。

### 快速部署（Ubuntu 22.04）

`ash
# 1. 上传项目到服务器
scp -r tax-automation user@server:/tmp/

# 2. SSH 登录服务器并运行部署脚本
ssh user@server
cd /tmp/tax-automation
sudo bash deploy/deploy.sh
`

部署完成后访问 `http://服务器IP` 即可使用。

### 本地开发启动

`ash
# Linux/Mac
bash deploy/start_local.sh

# Windows
python run_web.py
`

## 安装与运行

### 环境要求

- Python 3.8+
- Windows 10/11

### 安装步骤

1. 安装依赖：

```bash
pip install playwright
playwright install chromium
```

2. 启动程序：

```bash
cd D:\tax-automation
python main.py
```

## 使用说明

### 1. 添加经营主体

1. 点击【管理】→【经营管理主体】
2. 点击【添加主体】
3. 填写主体信息（名称、信用代码、类型、省份等）
4. 点击保存

### 2. 记账

1. 点击【管理】→【记账】
2. 选择经营主体和日期
3. 录入收入或支出明细
4. 系统自动更新季度汇总和税额预估

### 3. 快速录入收入数据

1. 点击【管理】→【快速收入录入】
2. 选择经营主体和季度
3. 输入季度收入和费用
4. 系统优先读取记账汇总并预览税额

### 4. 查看税额计算

1. 点击【工具】→【税额计算器】
2. 输入收入和费用数据
3. 点击【计算税额】查看结果

## 自动报税功能（实验性）

### 功能说明

系统现已支持福建省电子税务局自动报税功能（实验性），可以自动完成以下操作：

- 登录福建省电子税务局（支持手动登录、扫码登录）
- 自动填写增值税申报表
- 自动填写个人所得税（经营所得）申报表
- 自动填写附加税申报表
- 生成申报日志和截图

### 使用步骤

1. 点击左侧菜单【自动报税】
2. 选择经营主体
3. 选择省份（目前仅支持福建）和登录方式
4. 点击【登录】，在浏览器中完成登录
5. 选择需要申报的税种和申报期间
6. 点击【刷新数据】查看申报数据预览
7. 点击【开始自动申报】
8. 在浏览器中核对申报信息并确认提交

### 注意事项

- **实验性功能**：自动报税目前为实验性功能，建议在使用前备份数据
- **页面结构变化**：如果税务局网站页面结构发生变化，自动填写可能失效
- **验证码处理**：部分登录方式需要手动输入验证码或扫码
- **申报确认**：系统会自动填写申报表，但最终提交需要用户手动确认
- **操作日志**：所有申报操作都会记录在 `data/logs/` 目录

### 支持的省份

- 福建省（已实现）
- 其他省份待扩展


## 当前限制

1. **不是自动报税系统**：当前版本不会代替用户登录电子税务局并提交申报。
2. **自动化模块为实验性**：`automation/` 目录中的浏览器自动化与地区适配器仅作为未来扩展的示例骨架，当前不建议直接用于生产环境。
3. **企业查询依赖网页结构**：如果天眼查/爱企查页面结构变化，提取结果可能失效。
4. **数据全部本地存储**：数据库文件位于 `data/entities.db`，请自行备份。
5. **税率与规则为简化实现**：适用于常见小规模场景，复杂业务仍需人工核对。
